	Функциональные требования

1. Импорт списка сотрудников из текстового файла формата CSV.
2. Возможность выбора подразделений из справочника.
3. Сортировка элементов отчета для реестра. Визиты одного пациента группируются.
   Группа визитов пациента сортируется по возрастанию даты визиты. Затем 
   сортируются группы визитов пациентов во всей выборке. Группы сортируются по 
   возрастанию наибольших дат визита групп.
